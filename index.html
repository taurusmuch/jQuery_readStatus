ABOUT
此套件為提供使用者閱讀網頁時的瀏覽進度
$(this).readstatus(
1.載入jQuery
<script src="js/jquery-2.1.1.min.js"></script>
2.載入readStatus.js
<script src="js/jQuery_readStatus.js"></script>
3.啟動reasStatus
$(this).readstatus();
CONFIGURATION PARAMETERS
pic 圖片網址 String, 輸入想要的圖片
float 圖像顯示位置 String, left:左側;right:右側
appendTo 套件套用的位置 String, 預設位置為body
bottom 套件下方留白的高度 Integer, 預設高度10px
left 左(右)側留白的寬度 Integer, 預設寬度10px
maxWidth 圖片最大寬度 Integer, 預設為50px
maxHeight 圖片最大高度 Integer, 預設為50px
